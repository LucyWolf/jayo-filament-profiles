🧵 Jayo Filament Profiles — Anycubic Slicer
Community-tested filament profiles for Anycubic Slicer (Next).
Profiles are organized by slicer and filament type.
Each profile includes both the .json and .info file — always download both together!

⚠️ These profiles are tuned for the Anycubic Kobra S1 (Direct Drive).
Settings like retraction may need adjustment for Bowden printers.

📥 Installation

Download both the .json and .info file for your filament
Copy both files to:
C:\Users\<username>\AppData\Roaming\AnycubicSlicerNext\user\<id>\filament\
Restart Anycubic Slicer
Select the profile from the filament dropdown
